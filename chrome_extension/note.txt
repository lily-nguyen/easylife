Tomcat setup
1. clone Tomcat
2. Unzip Tomcat
3. Add Server into Eclipse: Window > Show Views > Servers > Apache > Tomcat v.8 > set Location
4. Start Tomcat (in Eclipse)
5. Copy $ApacheHome/webapps/ROOT into workspace/.metadata/.../wtpwebapps (replace)  
6. Restart Tomcat
